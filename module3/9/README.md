# POSIX Semaphores — Producer/Consumer (Processes)

## Описание задачи

Данная программа реализует взаимодействие двух процессов (родительского и дочернего) с использованием **именованных семафоров POSIX**.

- Родительский процесс генерирует строки из случайных чисел и записывает их в файл.
- Дочерний процесс считывает строки, находит в каждой строке min и max, и выводит результат на экран.
- Доступ к файлу синхронизируется с помощью семафора.

--- 
## Запуск программы 
- make
- ./prog "name_file.txt"

##  Используемые технологии

- C (C11)
- POSIX процессы (`fork`)
- POSIX именованные семафоры (`sem_open`, `sem_wait`, `sem_post`)
- Работа с файлами (`fopen`, `fgets`, `fprintf`)
- Linux (/dev/shm для семафоров)

---
- `sem_wait()` — захват ресурса
- `sem_post()` — освобождение ресурса
- `sem_close()` — закрытие семафора в процессе
- `sem_unlink()` — удаление семафора из системы

## ## Завершение программы

Программа корректно завершается по нажатию: CTRL + C
Используется обработка сигнала `SIGINT`:
- устанавливается флаг завершения
- процессы выходят из циклов
- ресурсы освобождаются
