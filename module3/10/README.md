# Разделяемая память System V 
## Описание задачи

Реализована программа на языке C, демонстрирующая использование механизма межпроцессного взаимодействия — разделяемой памяти System V.

Программа выполняет следующие действия:

- Родительский процесс генерирует набор случайных чисел.
- Дочерний процесс находит минимальное и максимальное значение.
- Результат передаётся обратно через разделяемую память.
- Родительский процесс выводит результат на экран.
- Работа продолжается до получения сигнала SIGINT (Ctrl+C).
- После завершения выводится количество обработанных наборов.

---

## Используемые технологии

- System V Shared Memory (shmget, shmat, shmdt, shmctl)
- fork() для создания дочернего процесса
- Сигналы (SIGINT)
- Busy-wait синхронизация (ready / done)

## Завершение программы

Для остановки нажмите:

Ctrl + C

После завершения программа:

- корректно завершит дочерний процесс
- освободит разделяемую память
- выведет количество обработанных наборов
