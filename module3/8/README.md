# Producer–Consumer (System V Semaphores)

##  Описание задания

Реализована модель "производитель–потребитель" с использованием семафоров System V.

## ВАЖНО ! 
Остановка программы Ctrl + C 
После необходимо:
1.Проверка семафоров ipcs -s
2.Удаление семафора ipcrm -s <semid>

### Producer:
- генерирует строки случайных чисел
- записывает их в файл
- помечает строки как необработанные (`| 0`)

### Consumer:
- читает файл
- находит необработанные строки (`| 0`)
- вычисляет минимальное и максимальное значение в строке
- помечает строку как обработанную (`| 1`)

## Синхронизация

Для защиты файла используется **System V semaphore**:

- `semid` создаётся через `semget()`
- доступ к файлу защищается операциями:
  - `P(mutex)` → semop(semid, -1)
  - `V(mutex)` → semop(semid, +1)
