# UDP чат (сокеты)

## Описание задачи

Реализована программа на языке C, демонстрирующая работу UDP-сокетов для обмена сообщениями между двумя клиентами.
Для реализации асинхронности использовался fork

Программа выполняет следующие действия:

- Создаёт UDP-сокет для каждого клиента
- Привязывает сокет к локальному порту (`bind`)
- Осуществляет обмен сообщениями между двумя клиентами
- Поддерживает асинхронное взаимодействие (отправка и получение сообщений в любом порядке)
- Разделяет логику отправки и приёма сообщений с помощью `fork()`
- Один процесс отвечает за отправку сообщений, второй — за получение
- Завершение работы происходит по сигналу `Ctrl + C`

---

## Используемые технологии

- UDP-сокеты (`SOCK_DGRAM`)
- IPv4 (`AF_INET`)
- Системные вызовы:
  - `socket()`
  - `bind()`
  - `sendto()`
  - `recvfrom()`
- `fork()` для параллельной обработки отправки и приёма
- Сигналы (`SIGINT`)
