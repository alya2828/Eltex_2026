# IPC Chat (System V)

## Описание

Простая программа чата на основе очередей сообщений System V.
Сервер принимает сообщения от клиентов и рассылает их другим клиентам.

---

## Файл `chat`

Файл `chat` используется только для генерации IPC-ключа через `ftok`.
Он не хранит сообщения.

Перед запуском:

```id="a1k9zp"
touch chat
```

---

## Сборка

```id="t9d2ks"
make
```

---

## Запуск

Сначала запустить сервер:

```id="o3l1vn"
./server
```

Затем клиентов (в разных терминалах):

```id="s8qp4h"
./client 1
./client 2
```

---

## Использование

* Вводите сообщение и нажимайте Enter
* Чтобы выйти, введите:

```id="n4v8xy"
shutdown
```
